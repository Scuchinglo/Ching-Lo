因为刚好在搭建博客所以在搭建过程中遇到了很多问题，所以我在这里把一些奇奇怪怪的问题总结下来。

##hexo deploy出现错误

最后一步hexo deploy时经常会遇见错误.

其原因是因为hexo版本太高

此时只需要手动安装模块即可

npm install hexo-deployer-git --save

##同时对于3.0版本的hexo

要把config文件里面的github改用git

再者就是对于deploy中的repo要用http，如果采用https同样会出现错误
